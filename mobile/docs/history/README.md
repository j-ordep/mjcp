# Historico de Documentacao

Esta pasta guarda materiais que continuam uteis como contexto historico, mas que nao devem mais competir com a documentacao viva da raiz de `docs/`.

## O que fica aqui

- registros de rodadas especificas com data no nome
- notas de hotfix
- prompts e rascunhos usados em etapas anteriores
- documentos de ideacao ou procedimentos operacionais pontuais que ja foram consolidados em outros arquivos

## O que fica na raiz de `docs/`

- `CONTEXT.md`
- `system_design.md`
- `scheduling_model.md`
- `ROADMAP.md`
- `TASKS.md`
- `NEXT_STEPS_PLAN.md`

Se algum arquivo historico voltar a representar regra atual do produto, ele deve ser consolidado em um dos documentos vivos acima.
